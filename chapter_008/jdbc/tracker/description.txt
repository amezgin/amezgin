Программа состоит из 2х пакетов.

Пакет models содержит 2 класса:
	Класс Item описывает модель заявки. 
	Класс состоит из приватных полей (name (Имя), description (Описание), 
	dateCreate (Дата создания), сomment (список комментариев), id (идентификатор заявки в системе)) 
	и методов позволяющих получать хранимые значения и устанавливать значения (геттеры и сеттеры).

	Класс Comment описывает модель комментария.
	Класс состоит из приватного поля comment (комментарий)
	и методов позволяющих получать хранимые значения и устанавливать значения (геттеры и сеттеры).

Пакет start содержит 2 класса:
	Класс StartTracker содержит точку входа (метод main) и методы,
	которые выводят описание программы (showDescription()) и меню (showMenu()).
	Описание программы хранится в приватном поле description.

	Класс Tracker используется для хранения заявок и работы с ними.
	Содержит список заявок - items и
	методы для работы с заявками и комментариями.
	addComment - добавить комментарий к заявке.
	addItem - добваить заявку.
	editItem - редактировать заявку.
	removeItem - удалить заявку.
	finfByName - найти заявку по имени.
	findById - найти заявку по идентификатору.
	findByDescription - найти заявку по описанию.	
	showAllItem - показать все заявки.
